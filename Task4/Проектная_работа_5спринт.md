SRE — Site Reliability Engineering (команда сопровождения)

Таблица ролей и групп пользователей 

| Роль                  | Права роли                                                                                                                                            | Группы пользователей                          |
|-----------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------|
| cluster-admin-prod    | Права на доступ ко всем ресурсам кластера (включая просмотр/редактирование секретов; создание/удаление namespaces; настройку RBAC; управление узлами) | platform:devops, platform:sre                 |
| cluster-viewer        | Права на просмотр всех ресурсов кластера (nodes, pods, services, configmaps, secrets без расшифровки данных). Без права редактирования                | platform:security-audit, platform:managers    |
| namespace-developer   | Права на создание, удаление, изменение pods, services, deployments, configmaps в своих namespaces. Нет доступа к secrets и RBAC                       | product:functional-team, product:backend-devs |
| namespace-viewer      | Права на чтение ресурсов внутри конкретных namespaces (просмотр логов, событий, статусов). Без доступа к secrets                                      | product:business-analysts, product:managers   |
| secret-reader         | Права на чтение секретов в кластере (в namespaces, с которыми работает сотрудник). Право необходимо для DevOps при отладке и для Security при аудите  | platform:devops, platform:security-audit      |
| integration-external  | Права на создание ServiceAccounts и получение токенов для внешних интеграций (партнёры). Без доступа к другим ресурсам                                | platform:integration-engineering              |
|                       |                                                                                                                                                       |                                               |


Обоснование групп пользователей

| Группа                            | Кто входит                                                       | Комментарий                                     |
|-----------------------------------|------------------------------------------------------------------|-------------------------------------------------|
| platform:devops                   | Инженеры по эксплуатации, DevOps                                 | Полный контроль над кластером                   |
| platform:sre                      | Группа сопровождения                                             | Восстановление, анализ и исправление инцидентов |
| platform:security-audit           | ИБ, аудиторы                                                     | Только просмотр, без изменений                  |
| platform:managers                 | Менеджеры продукта, руководители                                 | Просмотр состояния кластера для отчётности      |
| product:functional-team           | Разработчики                                                     | Работа в своих namespace                        |
| product:backend-devs              | Бэкенд-разработчики                                              | Работа в своих namespace                        |
| product:managers                  | Менеджеры внутри продукта                                        | Просмотр статусов                               |
| platform:integration-engineering  | Инженеры, отвечающие за интеграции с партнёрами (УК, умный дом)  | Создание ServiceAccount для внешних систем      |
|                                   |                                                                  |                                                 |
