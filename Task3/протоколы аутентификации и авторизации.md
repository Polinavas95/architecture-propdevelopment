| Компонент                                              | Протокол                    | Детали                                                                                                                     |
|--------------------------------------------------------|-----------------------------|----------------------------------------------------------------------------------------------------------------------------|
| Мобильное приложение -> API Gateway PropDevelopment    | OAuth2 + JWT (Bearer token) | Токен выдаётся после входа в мобильное приложение. Содержит id user_id, роль owner, время жизни access_time, refresh token |
| API Gateway → Smart Home Integration Service           | mTLS                        | Сервис-сервисная аутентификация внутри приватного облака                                                                   |
| Smart Home Integration Service → Партнёрская платформа | API Key                     | Партнёр не должен знать про конкретного пользователя. Только PropDevelopment вызывает API партнёра                         |
| Партнёрская платформа → устройства (домофон/шлагбаум)  | MQTT +  TLS                 | Требуется документальное подтверждение безопасности                                                                        |
